=====================================================
 FORMER REBELS DATABASE SYSTEM
 Provincial Social Welfare and Development Office
 Occidental Mindoro
 Version 1.0 — 2026
=====================================================

REQUIREMENTS
------------
- Windows 10 or 11
- Google Chrome or Microsoft Edge (latest version)
- Internet connection (required for Firebase sync)

INSTALLATION
------------
1. Right-click INSTALL.bat
2. Select "Run as administrator"
3. Follow the on-screen steps
4. A shortcut "FR DATABASE" will be created on your Desktop
   and in the Start Menu

LAUNCHING THE APP
-----------------
- Double-click the "FR DATABASE" shortcut on your Desktop, OR
- Double-click LAUNCH.bat from the installation folder, OR
- Open index.html directly in Chrome or Edge

DEFAULT LOGIN
-------------
  Username : ADMIN
  Password : (the password you set in Firebase Authentication)

NOTES
-----
- All data is stored in the cloud (Firebase / Google Firestore).
  Your records are safe even if you reinstall the app.
- Internet connection is required to load and save records.
- Photos and files are uploaded to Firebase Storage.
- Only admin@frdb.local has ADMIN access.
  All other accounts are OPERATOR (read/write records only).

UNINSTALL
---------
1. Delete the folder: C:\Program Files\FRDB
2. Delete the Desktop shortcut: FR DATABASE
3. Delete the Start Menu shortcut:
   %APPDATA%\Microsoft\Windows\Start Menu\Programs\FR DATABASE.lnk

SUPPORT
-------
For technical issues, contact your system administrator.
