=====================================================
 FORMER REBELS DATABASE SYSTEM
 Provincial Social Welfare and Development Office
 Occidental Mindoro
 Version 1.0 — 2026
=====================================================

APP URL
-------
  https://ocmfrdb.vercel.app

REQUIREMENTS
------------
- Windows 10 or 11
- Google Chrome or Microsoft Edge (latest version)
- Internet connection (required — app runs on the cloud)

INSTALLATION
------------
1. Double-click INSTALL.bat
   (No need to run as administrator)
2. Follow the on-screen steps
3. A shortcut "FR DATABASE" will be created on your Desktop
   and in the Start Menu

HOW IT WORKS
------------
The installer creates a shortcut that opens the app at
https://ocmfrdb.vercel.app in Chrome or Edge using
"--app mode" — this hides the browser address bar so it
looks and feels like a real desktop application.

All data is stored securely in Firebase (Google Cloud).

LAUNCHING THE APP
-----------------
- Double-click "FR DATABASE" on your Desktop, OR
- Search "FR DATABASE" in the Start Menu

DEFAULT LOGIN
-------------
  Username : ADMIN
  Password : (the password set in Firebase Authentication)

  Operator accounts are created by the ADMIN
  in the User Management tab.

NOTES
-----
- Internet connection is REQUIRED at all times.
- Data is stored in Firebase Firestore (cloud).
- Only admin@frdb.local has full ADMIN access.
- All other accounts are OPERATOR level.

UNINSTALL
---------
1. Delete Desktop shortcut: FR DATABASE
2. Delete Start Menu shortcut:
   %APPDATA%\Microsoft\Windows\Start Menu\Programs\FR DATABASE.lnk
3. Delete launcher folder:
   %USERPROFILE%\AppData\Local\FRDB\

SUPPORT
-------
For technical issues, contact your system administrator.
